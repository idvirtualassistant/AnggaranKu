Ringkasan

AnggaranKu adalah aplikasi pencatat pengeluaran & anggaran yang berjalan sepenuhnya offline. Data Anda disimpan di perangkat dan tidak dikirim ke server kami.

Data yang Dikumpulkan

Kami tidak mengumpulkan data pribadi ke server. AnggaranKu tidak mewajibkan akun dan tidak memiliki fitur login.

Data yang Disimpan di Perangkat

  • Transaksi pemasukan/pengeluaran (nominal, kategori, tanggal, catatan)
  • Anggaran (budget) per periode/kategori
  • Pengaturan aplikasi (mis. kategori, preferensi tampilan)

Pembagian Data

Kami tidak membagikan data kepada pihak ketiga karena data tidak dikirim keluar perangkat.

Izin Aplikasi

Jika aplikasi memerlukan izin tertentu (misalnya penyimpanan) itu hanya untuk fitur lokal seperti ekspor/backup file, dan tidak digunakan untuk mengirim data ke server.

Keamanan

Karena data tersimpan lokal, keamanan juga bergantung pada keamanan perangkat Anda. Jika tersedia, Anda dapat mengaktifkan fitur penguncian aplikasi (PIN/biometrik).
