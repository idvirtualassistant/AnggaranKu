Kebijakan Privasi

Terakhir diperbarui: 2026-01-22

Ringkasan: AnggaranKu adalah aplikasi pencatat keuangan pribadi yang berjalan sepenuhnya offline. Data transaksi disimpan di perangkat Anda dan tidak dikirim ke server kami.

1) Informasi yang Kami Kumpulkan

AnggaranKu tidak mengumpulkan data pribadi ke server. Aplikasi tidak mewajibkan akun dan tidak memiliki fitur login.

2) Informasi yang Disimpan di Perangkat

Untuk menjalankan fungsi aplikasi, AnggaranKu dapat menyimpan data berikut secara lokal di perangkat:

  • Transaksi pemasukan/pengeluaran (nominal, kategori, tanggal, catatan)
  • Anggaran (budget) per periode/kategori
  • Pengaturan aplikasi (mis. kategori, preferensi tampilan)

3) Cara Kami Menggunakan Informasi

Data digunakan hanya untuk menampilkan catatan keuangan dan laporan di aplikasi, serta membantu Anda mengelola anggaran. Karena aplikasi offline, penggunaan data terjadi di perangkat Anda.

4) Berbagi Informasi

Kami tidak membagikan data Anda kepada pihak ketiga karena data tidak dikirim keluar perangkat.

5) Izin Aplikasi

Jika aplikasi meminta izin tertentu (misalnya penyimpanan), izin tersebut hanya digunakan untuk fitur lokal seperti ekspor/backup file, dan bukan untuk mengirim data ke server.

6) Keamanan

Karena data tersimpan lokal, keamanan juga bergantung pada keamanan perangkat Anda. Jika tersedia di aplikasi, Anda dapat mengaktifkan fitur penguncian (PIN/biometrik).

7) Retensi Data

Data disimpan selama Anda menyimpannya di aplikasi. Anda dapat menghapus data kapan saja (lihat halaman Penghapusan Data).

8) Privasi Anak

AnggaranKu tidak dirancang khusus untuk anak-anak dan tidak sengaja mengumpulkan data pribadi anak.

9) Perubahan Kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu. Versi terbaru akan dipublikasikan di halaman ini.

10) Kontak

Jika ada pertanyaan tentang privasi, hubungi: editorial@virtualassistant.biz.id
