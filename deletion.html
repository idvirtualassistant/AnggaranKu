Penghapusan Data

Terakhir diperbarui: 2026-01-22

Data disimpan lokal (offline)

AnggaranKu tidak menyimpan data Anda di server. Semua data berada di perangkat.

Cara menghapus data

  1. Buka aplikasi AnggaranKu
  2. Masuk ke Pengaturan (jika tersedia)
  3. Pilih opsi Hapus Semua Data (jika tersedia)
  4. Atau uninstall aplikasi untuk menghapus data lokal dari perangkat

Catatan Backup / Ekspor

Jika Anda membuat file backup/ekspor (mis. CSV), file tersebut tersimpan di perangkat Anda. Anda dapat menghapusnya secara manual dari file manager.

Jika Anda membutuhkan bantuan terkait penghapusan data, hubungi: editorial@virtualassistant.biz.id
